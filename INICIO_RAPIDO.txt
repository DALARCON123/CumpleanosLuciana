# 🚀 INICIO RÁPIDO - 5 MINUTOS

Si tienes prisa, aquí está lo esencial para hacer funcionar la invitación.

---

## ⏰ En 5 Minutos

### 1️⃣ Abre la página (30 segundos)
```
Haz doble clic en: index.html
```

### 2️⃣ Cambia la foto (1 minuto)
```
1. Coloca tu foto en la carpeta: images/
2. Renómbrala como: luciana.jpg
3. Listo, aparecerá en la página
```

### 3️⃣ Cambia la fecha (1 minuto)
```
1. Abre index.html con un editor de texto
2. Busca: data-date="2026-05-23T18:00:00"
3. Cambia a tu fecha: 2026-06-15T18:00:00
4. Guarda el archivo
```

### 4️⃣ Cambia el número WhatsApp (1 minuto)
```
1. Abre js/script.js con un editor de texto
2. Busca: sendWhatsApp('+5491234567890')
3. Cambia el número: sendWhatsApp('+TU_NUMERO')
4. Guarda el archivo
```

### 5️⃣ Edita misa y recepción (1 minuto)
```
1. En la página, haz clic en cualquier texto editable
2. Modifica según necesites
3. Haz clic afuera para guardar
```

---

## 🎵 Bonus: Música (Opcional - 1 minuto)


## 💡 Tips Rápidos

💾 **Antes de cambiar código:**
- Duplica la carpeta como copia de seguridad
- Así tienes versión original si algo sale mal

🔍 **Cuándo edites:**
- Abre con editor de texto: Notepad++, VS Code (gratis)
- NO abras con Word (puede dañar el HTML)

🧹 **Si no ves cambios:**
- Presiona Ctrl + Shift + Delete (limpiar caché)
- O abre en modo incógnito

📸 **Para la foto:**
- Usa JPG (más liviano)
- Mínimo 250x350 píxeles
- Luciana debe estar centrada

🎵 **Para la música:**
- Busca música royalty-free
- Recomendado: instrumental, clásico, piano
- Máximo 5 MB
