## 📸 PASO 1: LA FOTO (MÁS IMPORTANTE)

Para que se vea hermoso:

1. **Busca una foto de Luciana:**
   - Tamaño: mínimo 250x350 píxeles (o más grande)
   - Formato: JPG o PNG
   - Calidad: clara y bien iluminada
   - Pose: Luciana mirando a la cámara, sonriendo

2. **Colócala en la carpeta `images/`:**
   - Haz clic derecho en la carpeta `images`
   - "Pegar aquí"
   - Renómbrala como: `luciana.jpg`

3. **Abre `index.html` en el navegador**
   - Doble clic en `index.html`
   - ¡La foto aparecerá automáticamente!

---

## 📅 PASO 2: LA FECHA (IMPORTANTE)

Para que funcione la cuenta regresiva:

1. **Abre `index.html` con un editor de texto**
   - Si no tienes: descarga Notepad++ (gratis) o VS Code (gratis)

2. **Busca esta línea (está cerca del final):**
   ```html
   data-date="2026-05-23T18:00:00"
   ```

3. **Cambia la fecha:**
   - Formato: `YYYY-MM-DDTHH:MM:SS`
   - Ejemplo: `2026-05-23T18:00:00` = 23 de mayo 2026, 6:00 PM

4. **Guarda el archivo** (Ctrl + S)

---

## 📱 PASO 3: WHATSAPP (IMPORTANTE)

Para que funcione el botón:

1. **Abre `js/script.js` con un editor de texto**

2. **Busca esta línea:**
   ```javascript
   sendWhatsApp('+5491234567890')
   ```

3. **Reemplaza con tu número:**
   - Incluye el código de país: `+54` (Argentina), `+52` (México), etc.
   - Ejemplo: `sendWhatsApp('+5491234567890')`

4. **Guarda el archivo** (Ctrl + S)

---

## ⛪ PASO 4: EDITAR MISA Y RECEPCIÓN

LA MANERA MÁS FÁCIL (sin código):

1. **Abre `index.html` en el navegador**

## 🆘 SI ALGO NO FUNCIONA

1. **La foto no se ve:**
   - ¿Está en la carpeta `images/`?
   - ¿Se llama exactamente `luciana.jpg`?
   - ¿Es un archivo JPG válido?

2. **El contador muestra números negativos:**
   - Verifica que la fecha sea futura
   - Busca `data-date` en `index.html`

3. **El WhatsApp no abre:**
   - ¿Incluiste el `+` al inicio?
   - ¿Sin espacios ni guiones?
   - ¿Con código de país?

4. **Se ve extraño después de editar:**
   - Limpia el caché: Ctrl + Shift + Delete
   - Abre en modo incógnito
   - Intenta otro navegador
